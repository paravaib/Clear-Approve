How it works

From sheet to inbox in seconds

  1. Open Clear Approve from the Extensions menu inside your Google Sheet.
  2. Select rows you want to share. Only the selected data is included.
  3. Automatic formatting turns small selections into clean email cards and larger selections into a PDF attachment.
  4. Click Send. Nothing is shared until you explicitly send it.

Event-driven

Auto-send when your data changes

Smart Triggers watch a column for value changes and fire an email the moment a condition is met.

  1. Pick a trigger column â€” for example, a "Status" or "Approval" column.
  2. Set the condition: equals, contains, greater than, less than, or not equal to a value you define.
  3. Choose your rows and recipients. When any cell in the trigger column changes to match, that row's snapshot is sent automatically.
  4. Each row fires once per transition, so recipients aren't spammed with duplicates.
